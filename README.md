
# مستودع تمارين ومشاريع تعليمية بلغة بايثون

شرح موجز
---------
هذا المستودع يحتوي على مجموعة من سكربتات بايثون التعليمية والتطبيقية الصغيرة (واجهات رسومية، ألعاب، تمارين صفية، أمثلة على الكائنات/الكلاسات)، موجهة للتعلم والممارسة وليس كتطبيق واحد متكامل. الملفات منظمة في مجلدين رئيسيين يمثلان سلسلتي دروس/تمارين: `Aeith 8 python` و `Nain 9 python`.

ماذا يقدم هذا المستودع؟
---------------------
- أمثلة واجهات رسومية باستخدام Tkinter (نموذج إدخال معلومات طالب).
- أمثلة ألعاب بسيطة باستخدام pygame (مثل لعبة الثعبان).
- تمارين وتطبيقات سطر أوامر لإدارة مستخدمين، قوائم أفلام، تحديات برمجية، وغيرها.
- شروحات صغيرة وملفات تدريبية مكتوبة باللغة العربية داخل ملفات السكربت.

بنية المجلدات (مختصر)
---------------------
- `Aeith 8 python/` — سكربتات أمثلة وتمارين (مثل `student_gui.py`, `buiplict.py`, `Safn7__Ultimate Challenge.py`).
- `Nain 9 python/` — دروس وملفات تعليمية و ألعاب (مثل `lesson.py`, `لعبة الثعبان.py`, ملفات turtle التعليمية).
> ملاحظة: هناك أسماء ملفات تحتوي مسافات وحروف عربية — عند التشغيل من الطرفية ضع المسار بين علامات اقتباس.

المتطلبات
---------
- Python 3 (موصى به Python 3.8+)
- مكتبة pygame مطلوبة فقط لتشغيل لعبة الثعبان:
  ```bash
  pip install pygame
مكتبة Tkinter عادة مضمّنة مع توزيعات CPython على معظم الأنظمة (Linux/macOS/Windows). إن لم تكن متاحة، ثبّتها عبر مدير الحزم الخاص بتوزيعتك.

ملاحظات وإرشادات
هذه سكربتات تعليمية مستقلة — لا توجد بنية حزمة موحدة أو نقطة دخول موحدة للمشروع.
يمكنك استخدامها كمصادر تعليمية أو كأساس لتطوير مشاريع أكثر تنظيماً (مثلاً فصل المنطق عن الواجهة، إضافة تخزين دائم، أو إنشاء حزم قابلة لإعادة الاستخدام).
يفضّل إضافة ملف requirements.txt وملف README.md (هذا الملف) في جذر المستودع لتنظيف تجربة المستخدم.
انتبه عند التعامل مع أسماء الملفات التي تحتوي مسافات أو أحرف غير ASCII — استخدم اقتباس المسار أو إعادة تسمية الملفات لتسهيل التشغيل عبر سطر الأوامر.
